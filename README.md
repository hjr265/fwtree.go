# Fwtree.go

Fwtree.go provides an implementation of [Fenwick tree](http://en.wikipedia.org/wiki/Fenwick_tree).

## Installation

Install Fwtree.go using the go get command:

    $ go get github.com/hjr265/fwtree.go

The only dependencies are the Go distribution.
